# 저시력자를 위한 원화 화폐 분류

-   Object Detection을 해결 하기 위한 yolov5 모델 사용
-   화폐에 있는 특징을 통해 화폐를 구별하기 위한 모델을 만들고 성능 개선
-

## 역할 분담

-   모델링(개인)

environments:

-   colab
-   yolov5
-   python

## 개발 과정

-   모델에 맞는 폴더 구조 확인
-   이미지 축소 비율에 맞춰 좌표값 변경
-   모델에 맞는 정보 추출/ 형식 변경
-   화폐당 하나의 클래스로 변경
-   모델 선택하고 학습
